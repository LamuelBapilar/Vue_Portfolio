🛠️ Installation & Setup

to open follow this steps
1. Clone the repository
 ```
git clone https://github.com/LamuelBapilar/Vue_Portfolio.git
 ```
2. Go to project folder
```
cd Vue_Portfolio
 ```
3. Install dependencies
```
npm install
 ```
4. Run the project
```
npm run serve
 ```
6. Upon reaching the site enter this credential to the login page
 ```
username: admin 
password: 123
 ```
 
Optional:
there is a website preview in this project which you can watch to see the website.
